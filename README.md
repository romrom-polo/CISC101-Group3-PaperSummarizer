# CISC101 Paper Summarizer

This repository contains system prompt and module architecture of CISC101 Final project : Research Paper Summarizer.

## Structure

- `system_prompt.md`: Main system prompt defining the behavior, inputs, outputs, and constraints of the summarizer.
- `modules/01_intake_setup.md`: Intake & setup module that defines how inputs will be processed.
- `modules/02_section_loop.md`: Section loop module that defines how each section of research paper will be fed & checked.
- `modules/03_guardrails.md`: Hallucination prevention measure.
- `modules/04_rendering_refinement.md`: Output format rules, checks output structure.
- `modules/05_equation_explainer.md`: Contains equation explainer guidelines.
- `modules/06_area_of_study_indicator.md`: Contains area of study identification guideline.
